Gym Website — Frontend Project
A responsive gym website built using HTML, CSS, and JavaScript. Features include a sticky navigation bar with mobile hamburger menu, hero section, membership pricing plans, trainer profiles with an interactive lightbox popup, programs section, smooth scroll animations, and a light/dark theme toggle. Deployed on Vercel.
Built with: HTML,CSS,JavaScript
Live Demo: https://gym-website-pxpw.vercel.app
